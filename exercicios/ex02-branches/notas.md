# Exercício 02 — Branches

Neste repositório, as branches foram usadas para separar mudanças de funcionalidades, documentação e correções.

## Branches planejadas
- feat/agenda-inicial
- docs/registra-exercicios
- fix/ajusta-resumo

Cada branch foi criada com um propósito específico para manter o histórico mais organizado.
## Objetivo
Separar cada alteração em uma branch diferente tornou o processo mais claro e organizado.
