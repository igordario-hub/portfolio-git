
# Exercício 01 — Commits Semânticos

Neste exercício, pratiquei o uso de commits no padrão Conventional Commits.

## Tipos usados
- feat
- fix
- docs
- chore
- refactor
- test

## Observação
A proposta foi registrar as mudanças de forma clara, usando mensagens que explicam a intenção de cada alteração.
